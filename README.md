# TryHackMe Learning Paths – Write-Ups & Notes

This repository contains my personal notes, solutions, and write-ups for TryHackMe learning paths.  
The goal is to document my progress in **cybersecurity, networking, and system administration** while building hands-on experience.

---

## Learning Paths Covered

### 1. Pre Security
Acquire the basic computer science skills required to get started in cybersecurity.  
**Key topics:**
- Cybersecurity basics (offensive & defensive)
- Networking fundamentals (LAN, OSI model, packets & frames)
- Web technologies and common attacks (DNS, HTTP, websites)
- Linux fundamentals (Part 1–3)
- Windows fundamentals (Part 1–3)

---

## 2. Cyber Security 101
Develop cybersecurity skills needed to enter any career in the industry.  
**Key topics:**
- Networking concepts, secure protocols, and tools (Wireshark, Tcpdump, Nmap)
- Cryptography basics (hashing, public key, John the Ripper)
- Linux, Windows, and Active Directory fundamentals
- Offensive security (Metasploit, SQLMap, Hydra, Gobuster)
- Web hacking (JavaScript, SQL, Burp Suite)
- Defensive security (SOC fundamentals, incident response, SIEM, firewalls, IDS)
- Security tooling (CyberChef, CAPA, REMnux, FlareVM)
- OWASP Top 10 (2025)



### 3. SOC Level 1
Get on the fast track to becoming a successful Security Analyst.  
**Key topics:**
- Security operations center fundamentals
- Digital forensics and incident response
- Log analysis and vulnerability scanning
- Practical SOC workflows and tooling


## Tools & Skills Practiced
- **Operating Systems**: Linux (Bash), Windows (CMD, PowerShell)  
- **Networking**: TCP/IP, LAN, OSI model, secure protocols  
- **Security Tools**: Metasploit, Burp Suite, Hydra, SQLMap, Wireshark, Tcpdump  
- **Defensive Tools**: SIEM, IDS, CyberChef, CAPA, REMnux, FlareVM  
- **Programming & Scripting**: Python, HTML, CSS, JavaScript  
- **Version Control**: Git/GitHub for documentation and project tracking
